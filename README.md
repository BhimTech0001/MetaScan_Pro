# MetaScan_Pro

A Python tool to:
- Extract EXIF metadata
- Scan for file signatures
- Compute file hashes (MD5, SHA-1, SHA-256)

## Requirements
- Python 3.7+

## Installation
Clone the repository:
git clone https://github.com/BhimTech0001/MetaScan_Pro.git
cd MetaScan_Pro
Python MetaScan_pr0.py
